## Setup
Download [Node.js](https://nodejs.org/en/download/).
Run this followed commands:

``` bash
# Install dependencies (only the first time)
npm install

# Run the local server at localhost:8080
npm run dev

# Build for production in the dist/ directory
npm run build
```
## A 3D interactive space with a scary house effect made with THREE.JS, WebGl and WebPack Bundler..

Geometries have been added with the use of geometries available in three.js itself, mutiple textures are used which is in static folder. Dat.Gui library has been used to add tweak controls in the page.
